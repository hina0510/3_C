.cln 파일은 실행가능
.c 파일은 실행안됨(코드 읽기만 가능)

ctrl + F7 컴파일(오류확인)
ctrl + F5 실행

변수 : 데이터를 저장하는 공간, 변수 이름 부여
(첫 글자는 소문자(test), 합성어의 첫 글자는 대문자(youName),
의미를 부여해서 만든다, 예약어는 사용하면 안된다(main, printf))

자료형 : 
int(정수형, 4byte),
float(실수형, 4byte),
double(실수형, 8byte),
char(문자형, 1byte), 

형변환 : 자료형을 변화시키는 것

정수 나누기 정수 = 정수결과(몫)

더미값(쓰레기값)이 나오는 이유 : 문자열의 끝에는 NULL값이 들어간다
NULL은 1byte, 문자열의 끝을 알려주는 용도
NULL = 0 = '\0' (뒤에 설명 165분부터)

printf("%5d\n", num);  =>%d 사이에 수 넣으면 수만큼 칸수 확보(-는 왼쪽 정렬, 양수는 오른쪽 정렬)
