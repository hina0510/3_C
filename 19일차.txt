정적 할당 : 컴파일(문법검사) 시 변수의 크기가 할당되는 것
	   stack 영역에 생성된다
	   함수의 호출이 일어나면 해당 공간이 생성되고, 함수가 종료되면 해당 공간 또는 자동으로 사라진다
		char name[20];

동적 할당 : 프로그램 실행 중(ctrl+f5) 변수의 크기가 할당되는 것
	   <stdlib.h>, <mlloc.h>	=>	malloc 함수를 사용하여 할당
	   Heap 영역에 생성된다
	   사용자가 원할 때 할당하고 다 사용 후 사용자가 해제해야 한다
		char* name = (char*)malloc(20);
