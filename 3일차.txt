산술연산자 : +, -, *, /, %(mod, 나머지값)

#include<stdio.h>
void main() {
	int n1 = 20, n2 = 3;
	printf("%d / %d = %d\n", n1, n2, n1 / n2);     =>몫
	printf("%d / %d = %f\n", n1, n2, n1 / (float)n2);   =>몫 소수점
	printf("%d %% %d = %d\n", n1, n2, n1 % n2);   =>나머지

관계연산자 : 참과 거짓 표현, 0은 거짓, 0을 제외한 모든 것(양수, 음수)은 참, C언어는 참의 범위가 너무 커서 1로 표현
<, >, <=, >=, ==, !=

복합대입연산자 : +=, -=, *=, /=

논리연산자 : 결과적으로 참 또는 거짓
&&(and), ||(or), !(not)
&&(and) : 모든 값이 참이여야만 참
||(or) : 하나만 참이면 참
!(not) : !참 = 거짓, 반전시켜준다.

증감연산자 : 결과적으로 자기자신을 1증가 또는 감소
n++, n--, ++n, --n

char ch;
printf("문자 하나 입력 : ");
scanf_s("%c", &ch, 1);   =>문자일 경우 마지막에 입력받을 바이트 수 지정
printf("%c\n", ch);
char str[20];
printf("문자열 입력 : ");
scanf_s("%s", str, 20);   =>문자열일 경우 변수(=주소)를 받음
